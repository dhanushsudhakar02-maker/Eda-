# Eda-
## Project Overview
This project performs Exploratory Data Analysis on a Student Performance dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Statistical Summary
- Data Visualization
- Correlation Analysis

## Output
- Gender Distribution Chart
- Math Score Distribution
- Correlation Heatmap
